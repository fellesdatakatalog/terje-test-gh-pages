# ApplicationDCAT-AP-NO Spesifikasjon for beskrivelse av tilgangssøknader. ## Oppsummering Navnerommet til ApplicationDCAT-AP-NO er `https://raw.githubusercontent.com/Informasjonsforvaltning/applicationdcat-ap-no/refs/heads/develop/` Prefixet til ApplicationDCAT-AP-NO er `fdkapplication` ## Vokabular ### Datasett - tilgangsforespørselsURL
RDF Property: fdkapplication:accessRequestURL
Anvendelse/Usage note: Egenskapen brukes til å referere til en side eller tjeneste som håndterer forespørsel om tilgang til datasettet.
This property is used to refer to a webpage or service that handles a request for access to the dataset.
Sub-property of: dct:accessURL
Verdiområde/Range: rdfs:Resource
English name: Access request URL
